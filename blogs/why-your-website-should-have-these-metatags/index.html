Know about the different metatags and improve the SEO of your website in minutes.

Meta tags data helps to show the website preview in a visual way instead of just a plain old link whenever someone shares a link on social media, in messaging, or in your business chat software. Meta tags are also useful in search engines too.

Title tag

Title tags are displayed on search engine results pages as clickable headlines, and it's important for usability, SEO, and social sharing. So you must be careful to decide your title.

<title>Connect Oracle DB from Lambda Using Node.js</title>

Meta description

Sometimes search engines such as Google, displays the meta description in search results. It is a less important meta tag.

<meta name="description" content="In this article, You can learn how to connect to Oracle DB using the library 'oracledb-for-lambda'.">

Facebook / Open Graph Tags

Below are the meta tags that need to be added for facebook / open graph to improve the preview of your webpage.

Use this facebook tool to verify the preview of your website when shared in facebook. https://developers.facebook.com/tools/debug/

<meta property="og:url" content="https://geekyturtles.com/blogs/connect-oracle-db-from-lambda-using-node-js">
<meta property="og:type" content="website">
<meta property="og:title" content="Connect Oracle DB from Lambda Using Node.js">
<meta property="og:description" content="In this article, You can learn how to connect to Oracle DB using the library 'oracledb-for-lambda'.">
<meta property="og:image" content="http://geekyturtles.com/assets/static/cover.872e63c.ff0fede89ad5e86e6763ddae8b7af21c.png">

Twitter Meta Tags

Below are the meta tags that need to be added exclusively for twitter to improve the preview of your webpage when shared in twitter.

Use this tool to verify the preview of your website in official twitter. https://cards-dev.twitter.com/validator

<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Connect Oracle DB from Lambda Using Node.js">
<meta name="twitter:description" content="In this article, You can learn how to connect to Oracle DB using the library 'oracledb-for-lambda'.">
<meta name="twitter:image" content="http://geekyturtles.com/assets/static/cover.872e63c.ff0fede89ad5e86e6763ddae8b7af21c.png">
